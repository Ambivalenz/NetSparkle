Welcome to NetSparkle's website!

NetSparkle is a C# software update framework for .NET developers compatible with .NET Core, WinForms, WPF, and Avalonia. It can use either Ed25519 or DSA signatures!

Click here to view code documentation! For basic usage instructions and examples, please see the NetSparkle repository.
